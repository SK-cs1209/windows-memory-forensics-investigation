## 📄 Project Overview

This project demonstrates the analysis of a Windows memory dump to:

- Identify active and suspicious processes  
- Extract running processes and connections  
- Investigate potential attacker activity  
- Gain hands-on experience in memory forensics

---

## 📂 Deliverables

- Detailed analysis documented in [Report.docx](Report.docx)  
- Commands and methodology explained clearly

## 🎯 Learning Outcomes

- Understanding Windows memory structure  
- Hands-on experience with Volatility and CLI tools  
- Practical knowledge of detecting and investigating malicious activity

## 🔗 Goal

Develop expertise in digital forensics, memory analysis, and practical cybersecurity investigations while building a strong foundational skillset in penetration testing and incident response.
